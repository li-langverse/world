# Security policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a vulnerability

Report security issues to the [li-langverse](https://github.com/li-langverse) maintainers via GitHub Security Advisories on [`li-language`](https://github.com/li-langverse/li-language) or this package repository.

Do not open public issues for exploitable parser or proof-gate bypasses before coordinated disclosure.
