# li-world

PH-GD-2 GameWorld stub — scene graph hooks for World Studio compose (`workload_class=stub`; save/load text deferred).

## Build

```bash
lic build src/lib.li -o li-world
```

From the monorepo root, ensure `lic` is built: `./scripts/build.sh`.
