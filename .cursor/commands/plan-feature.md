# Plan feature from issue

Read and follow the skill **plan-feature-from-issue** (`.cursor/skills/plan-feature-from-issue/SKILL.md`).

1. Load the current GitHub issue or user description.
2. Read [vision-and-roadmap](https://github.com/li-langverse/roadmap/blob/main/docs/ecosystem/vision-and-roadmap.md) and the **lic** master plan PH tracker.
3. Produce a plan document (correct repo/path) with PH-*, tests, G-* gaps, Learned from.
4. Do **not** implement code until `plan-approved`.

Output the plan markdown in chat and list the exact file path to add in a PR.
